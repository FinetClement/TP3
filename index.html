Article sur JavaScprit

JavaScript est un langage de programmation de scripts principalement employé dans les pages web interactives mais aussi pour les serveurs2 avec l'utilisation (par exemple) de Node.js3.
C'est un langage orienté objet à prototype, c'est-à-dire que les bases du langage et ses principales interfaces sont fournies par des objets qui ne sont pas des instances de classes, mais qui sont chacun équipés de constructeurs permettant de créer leurs propriétés, et notamment une propriété de prototypage qui permet d'en créer des objets héritiers personnalisés.
En outre, les fonctions sont des objets de première classe. Le langage supporte le paradigme objet, impératif et fonctionnel. JavaScript est le langage possédant le plus large écosystème grâce à son gestionnaire de dépendances npm, avec environs 500 000 paquets en août 20174. JavaScript a été créé en 1995 par Brendan Eich. Il a été standardisé sous le nom d'ECMAScript en juin 1997 par Ecma International dans le standard ECMA-262. Le standard ECMA-262 en est actuellement à sa 8e édition. JavaScript n'est depuis qu'une implémentation d'ECMAScript, celle mise en œuvre par la fondation Mozilla. L'implémentation d'ECMAScript par Microsoft (dans Internet Explorer jusqu'à sa version 9) se nomme JScript, tandis que celle d'Adobe Systems se nomme ActionScript.

HTML

L'HyperText Markup Language, ce qui signifie en francais "langage de balisage d'hypertexte", désigne un type de langage informatique. Ce langage est utilisé pour créer des pages web. Cette signification porte bien son nom puisqu'effectivement ce langage permet de réaliser de l'hypertexte à base d'une structure de balisage.Ce n'est pas à proprement parlé un langage de programmation, mais plutôt un langage qui permet de mettre en forme du contenu. Les balises permettent de mettre en forme le texte et de placer des éléments interactif, tel des liens, des images ou bien encore des animations. Ces éléments ne sont pas dans le code source d'une page codé en HTML mais "à coté" et la page en HTML ne fait que reprendre ces éléments. Pour visualiser une page en HTML il est nécessaire d'utiliser un navigateur web.

Source : Tutorial HTML

SQL - Structured Query Language

Le SQL (Structured Query Language) est un langage permettant de communiquer avec une base de données. Ce langage informatique est notamment très utilisé par les développeurs web pour communiquer avec les données d’un site web. Par exemple, pour stocker les informations des utilisateurs (identifiants, mot de passe etc...)
Il est aujourd'hui reconnu par la plupart des SGBD (Système de Gestion de Base de Données)
Ce travail est souvent combiné avec d'autres langages tels que le PHP et l'HTML.
On retrouve beaucoup de tutoriels sur internet pour utiliser et compléter une base de données, par exemple : sql.sh, W3Schools, Developpez.com. Sachant que ce langage est composé de commandes "simple" il est très important d'avoir un bon sens de la structure et de l'organisation pour gérer au mieux la base de données

Source : sql.sh & Wikipedia

Le langage PHP

Qu'est ce que PHP?

PHP (officiellement, ce sigle est un acronyme récursif pour PHP Hypertext Preprocessor) est un langage de scripts généraliste et Open Source, spécialement conçu pour le développement d'applications web. Il peut être intégré facilement au HTML.

Qu'elle est la différence entre le PHP et le JavaScript?

Ce qui distingue PHP des langages de script comme le Javascript, est que le code est exécuté sur le serveur, générant ainsi le HTML, qui sera ensuite envoyé au client (Apache). Le client ne reçoit que le résultat du script, sans aucun moyen d'avoir accès au code qui a produit ce résultat. Vous pouvez configurer votre serveur web afin qu'il analyse tous vos fichiers HTML comme des fichiers PHP. Ainsi, il n'y a aucun moyen de distinguer les pages qui sont produites dynamiquement des pages statiques.

Les avantages d'utiliser PHP?

Le grand avantage de PHP est qu'il est extrêmement simple pour les néophytes, mais offre des fonctionnalités avancées pour les experts. Ne craignez pas de lire la longue liste de fonctionnalités PHP. Vous pouvez vous plonger dans le code, et en quelques instants, écrire des scripts simples.

Source : php.net

CSS

Définition

Les feuilles de style en cascade dans le codage, sont généralement appelés CSS. Il est utilisé pour mettre en forme des fichiers HTML ou XML, elles permettent donc de créer le design d'une page HTML. De nos jours il est plus judicieux d'utiliser une feuille de style plutôt que de décrire le design dans la page HTML, grâce a cette page il est possible de modifier plusieurs titres, plusieurs images à la fois plutôt que de les changer une par une dans le code HTML. Il est possible aussi de créer une feuille de style spécifique pour l'impression de document , ce qui permet de retirer tout les effets du style et toute les partie inutile lors de l'impression. De plus une feuille de style peut être utilisée par les utilisateur d'un téléphone portable ce qui permet de mieux gérer la mise en forme particulière aux petits écrans.

Wordpress : la facilité ?

Définition : Wordpress est un CMS (Content Management System) qui permet de contruire des sites web sans avoir besoin de trop s'y connaître en langage Web. De nombreuses entreprises ont fait confiance à ce CMS et l'utilisent régulièrement. C'est le cas du New York Times, Best Buy et Mashable pour ne citer qu'eux.

Les avantages : Wordpress possède de nombreux avantages le permettant de se hisser dans le top du classmenent des CMS utilisé. Un des grands avantages de WordPress est qu’il est très facile et très rapide à installer. C'est aussi un CMS relativement agréable à utiliser car très ergonomique. Enfin, Wordpress est plus complet que d'autres CMS en terme de plugins et de référencement.

Les inconvénients : Néanmoins, Wordpress n'est pas exempt de défauts. Tout d'abord, le fait que ce soit un CMS ne permet pas d'avoir le contrôle sur tout. Un site web plus travaillé peut permettre plus de choses que Wordpress. De plus, Wordpress n'est pas réputé pour sa sécurité. En effet, les CMS sont souvent ceux qui sont le plus en proie au piratage. Enfin, dernier point négatif, il ne vaut mieux pas installer énormément de plugins sur son site, sinon la vitesse de chargement de votre site risque de diminuer.
