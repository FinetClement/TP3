CSS

Définition

Les feuilles de style en cascade dans le codage, sont généralement appelés CSS. Il est utilisé pour mettre en forme des fichiers HTML ou XML, elles permettent donc de créer le design d'une page HTML. De nos jours il est plus judicieux d'utiliser une feuille de style plutôt que de décrire le design dans la page HTML, grâce a cette page il est possible de modifier plusieurs titres, plusieurs images à la fois plutôt que de les changer une par une dans le code HTML. Il est possible aussi de créer une feuille de style spécifique pour l'impression de document , ce qui permet de retirer tout les effets du style et toute les partie inutile lors de l'impression. De plus une feuille de style peut être utilisée par les utilisateur d'un téléphone portable ce qui permet de mieux gérer la mise en forme particulière aux petits écrans.
